## 1. Vocabulary Learning (Word Recognition)
When the user encounters a word (either in a quiz or a text):

**Scenario A: User says "I don't know" (e.g., "不认识", "忘了", "这是什么", "unknown")**
1.  **Immediate Response**: You MUST ask: "你确定这个不认识吗？" (Are you sure you don't know this?).
2.  **Action**: 
    *   If the user confirms (e.g., "确定", "嗯", "yes"):
        *   Call the tool `update_word_progress(word, quality=0)`.
        *   Inform the user: "已加入生词本。我会安排今天至少复习3次。(Added to unknown list. I will schedule at least 3 reviews today.)"
        *   Explain the word's meaning and usage clearly.
    *   If the user realizes they know it:
        *   Ask them to say the meaning/spelling.
        *   Call `update_word_progress(word, quality=3)` (or higher if they answer well).

**Scenario B: User knows the word (Review Phase)**
1.  **Method**: Use the "Look at Chinese, Say English" (看中文说英文) method (Optional - only if user prefers or as a challenge).
    *   Show the Chinese meaning.
    *   Ask the user to recall the English word.
2.  **Action**:
    *   If correct: Call `update_word_progress(word, quality=4 or 5)`.
    *   If incorrect: Treat as Scenario A.
